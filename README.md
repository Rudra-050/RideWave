🌊RideWave — Premium Ride-Hailing App
RideWave is a visually stunning, fully interactive ride-hailing web application built with vanilla HTML, CSS, and JavaScript. It simulates the complete experience of a service like Uber or Ola.

✨ Project Highlights
Modern Glassmorphism UI: A dark theme with vibrant accents (Electric Indigo, Cyan, Amber).
Interactive City Map: A custom CityMap class using the HTML5 Canvas API to simulate a city grid, with animated car movements and real-time route pathing.
Full Booking Flow: A multi-screen experience from the landing page to historical ride logs.
Micro-Animations: Extensive use of CSS animations for state transitions, loading indicators, and pulse effects.
🚀 Key Features
1. Landing Page
Hero Section: High-impact visuals with a data-driven stats counter.
Quick Book: Instant access to the booking flow directly from the hero.
Vehicle Showcase: Details on the four ride types: Go, Moto, XL, and Premier.
2. Booking Screen
Real-time Map Integration: The map updates instantly as pickup and drop-off locations are selected.
Intelligent Autocomplete: Mock location search for city landmarks.
Dynamic Pricing: Fare is calculated based on distance and selected vehicle type.
3. Driver Matching
Radar Scanning Animation: A beautiful pulsing radar that simulates searching for nearby drivers.
Driver Profile Reveal: Once a match is found, the driver's details (rating, vehicle, license plate) are displayed.
4. Trip Tracking
Animated Map: Watch the car move along the route on the city map.
Progress Stepper: Real-time status updates (En Route, Arrived, In Trip, Done).
Estimated Arrival: Live ETA calculations.
5. Ride History & Profile
Past Rides: View a list of previous trips with status and fare details.
User Dashboard: Manage wallet balance and account settings.
